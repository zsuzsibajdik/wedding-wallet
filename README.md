# Wedding Wallet

## About The Project
Wedding Wallet is a web application developed as a team project during the second (web) module at Codecool.

The purpose of the application is to support wedding planning by helping users track and manage expenses in a clear and structured way. Users can monitor their budget, keep track of booked services, and save potential vendors for future decisions. The goal is to make the financial side of wedding planning more transparent and easier to manage.

---

## Core Features
- User authentication using Firebase
- Expense tracking for wedding planning
- Manage booked services
- Track potential (not yet booked) services
- Real-time data handling with Firebase
- Responsive user interface built with React

---

## Built With / Tech Stack
- JavaScript
- React
- Firebase (Authentication & Database)
- HTML5
- CSS3
